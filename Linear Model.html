线性模型

给定一系列样本x(j)=(x1(j);x2(j),…,xn(j))x^{(j)} = (x_{1}^{(j)};x_{2}^{(j)},\dots,x_{n}^{(j)})x(j)=(x1(j)​;x2(j)​,…,xn(j)​),其中x1(j),x2(j),…,xn(j)x_{1}^{(j)},x_{2}^{(j)},\dots,x_{n}^{(j)}x1(j)​,x2(j)​,…,xn(j)​为第jjj个样本的nnn个属性值,线性模型尝试学习得到一个通过线性组合来进行预测的函数.即

f(x)=ω1x1+ω2x2+⋯+ωnxn+bf(x) = \omega_1 x_1 + \omega_2 x_2 + \dots + \omega_n x_n + bf(x)=ω1​x1​+ω2​x2​+⋯+ωn​xn​+b

用向量表示为

f(x)=ωTx+bf(x) = \omega^T x + bf(x)=ωTx+b

其中ω=(ω1;ω2;… ;ωn)\omega = (\omega_1; \omega_2 ; \dots ; \omega_n)ω=(ω1​;ω2​;…;ωn​).ω和b\omega 和 bω和b 通过学习得到后,模型就得以确定.

线性回归(linear regression)

线性回归是最基本的线性模型.给定数据集D={(x(j),y(j))∣1≤j≤m}D = \{(x^{(j)},y^{(j)}) | 1 \le j \le m\}D={(x(j),y(j))∣1≤j≤m}.线性模型试图学会

f(x(j))=ωx(j)+b≈y(j)f(x^{(j)}) = \omega x^{(j)} + b \approx y^{(j)}f(x(j))=ωx(j)+b≈y(j)

为了使预测准确,我们可以使和方差(SSE)最小化.

(ω∗,b∗)=argmin∑j=1m(f(x(j))−y(j))2=argmin∑j=1m(ωx(j)+b−y(j))2(\omega^*,b^*) = arg min \sum_{j=1}^{m} (f(x^{(j)}) - y^{(j)})^2 \\ = argmin \sum_{j=1}^{m} (\omega x^{(j)} + b - y^{(j)})^2(ω∗,b∗)=argminj=1∑m​(f(x(j))−y(j))2=argminj=1∑m​(ωx(j)+b−y(j))2

数学上求解上述表达式可以采用"正规方程法"."正规方程法"利用最小二乘法,试图找到一个超平面,使所有样本到平民啊地欧氏距离之和最小.

超平面:对于n维欧式空间,n-1维线性空间(余维数为1)被认为是一张超平面.

对于含有偏置bbb的回归模型,可以看成拥有n+1n+1n+1个属性的无偏置的线性模型.此时X矩阵可以写为

X=(x1(1)x2(1)…xn(1)1x1(2)x2(2)…xn(n)1⋮⋮⋱⋮⋮x1(m)x2(m)…xn(m)1)X = \begin{pmatrix} x^{(1)}_1 & x^{(1)}_2 & \dots & x^{(1)}_n & 1 \\ x^{(2)}_1 & x^{(2)}_2 & \dots & x^{(n)}_n & 1 \\ \vdots & \vdots & \ddots & \vdots & \vdots \\ x^{(m)}_1 & x^{(m)}_2 & \dots & x^{(m)}_n & 1 \\ \end{pmatrix}X=​x1(1)​x1(2)​⋮x1(m)​​x2(1)​x2(2)​⋮x2(m)​​……⋱…​xn(1)​xn(n)​⋮xn(m)​​11⋮1​​

目标写为矩阵

y=(y(1)y(2)⋮y(m))y = \begin{pmatrix} y^{(1)} \\ y^{(2)} \\ \vdots \\ y^{(m)} \end{pmatrix}y=​y(1)y(2)⋮y(m)​​

设ω^=(ω,b)\hat{\omega} = (\omega , b)ω^=(ω,b),此时,回归模型可以写成

y=X(ω,b)=Xω^y = X (\omega,b) = X \hat{\omega}y=X(ω,b)=Xω^

两边同时左乘XTX^TXT,得

XTy=XTXω^X^Ty = X^TX\hat{\omega}XTy=XTXω^

两边同时左乘(XTX)−1(X^TX)^{-1}(XTX)−1(伪逆),得

ω^=(XTX)−1XTy\hat{\omega} = (X^T X)^{-1} X^T yω^=(XTX)−1XTy

为最佳近似解.

除了正规方程法求解之外,还可以使用梯度下降的优化方法得到一个较优解.

设E(ω,b)=∑j=1m(ωx(j)+b−y(j))2E(\omega,b) = \sum_{j=1}^{m} (\omega x^{(j)} + b - y^{(j)})^2E(ω,b)=∑j=1m​(ωx(j)+b−y(j))2,我们分别对ω和b\omega和bω和b求偏导.

∂E∂ω=2(ω∑j=1mx(j)2−∑j=1m(y(j)−b)x(j))∂E∂b=2(mb−∑j=1m(y(j)−ωx(j)))\frac{\partial{E}}{\partial \omega} = 2 \bigg(\omega \sum_{j=1}^m {x^{(j)}}^2 - \sum_{j=1}^m(y^{(j)} - b) x^{(j)} \bigg) \\ \frac{\partial E}{\partial b} = 2 \bigg( mb - \sum_{j=1}^m (y^{(j)} - \omega x^{(j)}) \bigg)∂ω∂E​=2(ωj=1∑m​x(j)2−j=1∑m​(y(j)−b)x(j))∂b∂E​=2(mb−j=1∑m​(y(j)−ωx(j)))

令上式均为0即可得到ω\omegaω的最优解.

在实际的梯度下降中,受到各种参数的影响,得到的解可能与最优相差一定的距离.

示例代码

正规方程法

from FastModel.LR import *
from FastModel.GD import *
import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D
# =============================================================================
# 线性回归 -- 正规方程法
# =============================================================================
# 测试函数
f = lambda x,y : 3*x+2*y-2

func = lambda x: f(x[0],x[1])
#生成回归拟合集

x = np.linspace(-5,5,100)
y = np.linspace(-5,5,100)

x1 = np.random.uniform(-5,5,30)
y1 = np.random.uniform(-5,5,30)
z_noise = f(x1,y1) + np.random.randint(-5,5,size=x1.shape)

# 正规方程-线性回归求解器
X = np.vstack([x1,y1]).T
Y = z_noise.reshape(-1,1)
sol = NE_LSolver()
res = sol.fit(X,Y)
print("拟合的平面是 \n(%f) x + (%f) y + (%f)"%(res[1],res[2],res[0]))
prediction = sol.predict(X)


# 绘制网格 
xx,yy = np.meshgrid(x,y)
zz = f(xx,yy)

plt3d = plt.figure().gca(projection='3d')
plt3d.plot_surface(xx,yy,zz,alpha=0.2)

# 绘制预测点和原始点
ax = plt.gca()
ax.scatter(X[:,0],X[:,1],Y,color='r',alpha=0.8,label="origin")
ax.scatter(X[:,0],X[:,1],prediction,color='g',alpha=0.8,label="predict")

ax.set_xlabel("x")
ax.set_ylabel('y')
ax.set_zlabel('z')

ax.legend()

plt.show()

#拟合的平面是 
#(2.822226) x + (2.002309) y + (-2.885885)

线性回归变种

在真实世界中,自变量和因变量的关系可能并非线性关系,但可以通过一定的处理手段将其化为线性关系.例如1y=ωTx+b\frac{1}{y} = \omega^Tx+by1​=ωTx+b , ln(y)=ωTx+bln(y) = \omega^Tx+bln(y)=ωTx+b或者y=ω2Tx2+ω1Tx+by = \omega_2^Tx^2+\omega_1^Tx+by=ω2T​x2+ω1T​x+b.使用换元的思维,可以得到各种线性回归的变种(称为广义线性模型),例如逻辑回归,Softmax回归,多项式回归等.后面将会着重讲解这几种回归.

逻辑回归(Logistic Regrssion)

逻辑回归是一种线性回归的变种,通过对数几率函数(Sigmoid的一种)实现对样本二分类归属的回归和预测.

对于一个二分类的任务,不妨将输出限制在(0,1)(0,1)(0,1)内,如果一个样本归属于第一类,则希望它的输出预测值尽可能贴近0,否则应尽可能贴近1.

对数几率函数便可以满足这样的要求:

y=11+e−z(2.1)y = \frac{1}{1+e^{-z}} \tag{2.1}y=1+e−z1​(2.1)

其中z=ωTx+bz = \omega^Tx+bz=ωTx+b , 即:

y=11+e−(ωTx+b)(2.2)y = \frac{1}{1+e^{-(\omega^Tx+b)}} \tag{2.2}y=1+e−(ωTx+b)1​(2.2)

经过变换得到:

lny1−y=ωTx+b(2.3)ln \frac{y}{1-y} = \omega^Tx+b \tag{2.3}ln1−yy​=ωTx+b(2.3)

若将yyy看作样本xxx作为正例(第一类)的可能性,则1−y1-y1−y是其反例(第二类)可能性,两者之比称为"几率"(odds),对几率取对数,就得到了"对数几率"(logodds ,also called logit).

当然我们可以使用普通的线性回归对lny1−y和xln \frac{y}{1-y} 和 xln1−yy​和x进行回归,除此之外还可以使用"极大似然估计"来估计ω和b\omega和bω和b.

如果将对数几率函数中的yyy视为类后验概率估计p(y=1∣x)p(y = 1 | x)p(y=1∣x),则可以改写为:

lnp(y=1∣x)y=0∣x=ωTx+b(2.4)ln \frac{p(y = 1 | x)}{y = 0 | x} = \omega^Tx + b \tag{2.4}lny=0∣xp(y=1∣x)​=ωTx+b(2.4)

显然有:

p(y=1∣x)=11+e−(ωTx+b)(2.5)p(y= 1 | x) = \frac{ 1 }{1 + e^{-(\omega^Tx + b)}} \tag{2.5}p(y=1∣x)=1+e−(ωTx+b)1​(2.5)

p(y=0∣x)=1−11+e−(ωTx+b)(2.6)p(y=0 | x) = 1 - \frac{ 1 }{1 + e^{-(\omega^Tx + b)}} \tag{2.6}p(y=0∣x)=1−1+e−(ωTx+b)1​(2.6)

计算似然函数得:

L(ω,b)=∏j=1mp(y=1∣x)y(j)p(y=0∣x)(1−y(j))(2.7)L(\omega,b) = \prod_{j=1}^mp(y=1|x)^{y^{(j)}}p(y=0|x)^{(1-y^{(j)})} \tag{2.7}L(ω,b)=j=1∏m​p(y=1∣x)y(j)p(y=0∣x)(1−y(j))(2.7)

取对数得对数似然函数:

L′(ω;b)=∑j=1m[y(j)log(p(y=1∣x))+(1−y(j))log(p(y=0∣x))](2.8)L'(\omega;b) = \sum_{j=1}^m[y^{(j)}log(p(y=1|x)) + (1 - y^{(j)})log(p(y=0|x))] \tag{2.8}L′(ω;b)=j=1∑m​[y(j)log(p(y=1∣x))+(1−y(j))log(p(y=0∣x))](2.8)

我们只需要将对数似然函数最大化即可估计ω和b\omega 和 bω和b.

对L′L'L′取负号,即为二分类的交叉熵损失函数:

L(ω,b)=−∑j=1m[y(j)log(p(y=1∣x))+(1−y(j))log(p(y=0∣x))](2.9)L(\omega,b) = -\sum_{j=1}^m\bigg[y^{(j)}log(p(y=1|x)) + (1 - y^{(j)})log(p(y=0|x))\bigg] \tag{2.9}L(ω,b)=−j=1∑m​[y(j)log(p(y=1∣x))+(1−y(j))log(p(y=0∣x))](2.9)

或者写成常见的形式:

L(ω,b)=−∑j=1m[y(j)log(y(j)^)+(1−y(j))log(1−y(j)^)](2.10)L(\omega,b) = - \sum_{j=1}^m \bigg[ y^{(j)}log(\hat{y^{(j)}}) + (1-y^{(j)}) log(1- \hat{y^{(j)}}) \bigg] \tag{2.10}L(ω,b)=−j=1∑m​[y(j)log(y(j)^​)+(1−y(j))log(1−y(j)^​)](2.10)

使用梯度下降法降低交叉熵损失函数值,从而可以估计出参数ω,b\omega,bω,b的大小.

多分类学习

现实中的分类问题常常需要进行多分类.考虑NNN个类别C1,C2,…,CNC_1,C_2,\dots,C_NC1​,C2​,…,CN​,多分类学习的基本思路是拆分法,将多分类任务拆分为若干个二分类任务求解.

最经典的拆分策略有三种:"一对一"(One vs. One , OvO),"一对余"(One vs. Rest, OvR)和"多对多"(Many vs. Many , MvM).

OvO

给定数据集D={(x(1),y(1)),(x(2),y(2)),…,(x(m),y(m))},y(j)∈{C1,C2,…,CN}D = \{(x^{(1)},y^{(1)}),(x^{(2)},y^{(2)}),\dots,(x^{(m)},y^{(m)})\} , y^{(j)} \in \{ C_1,C_2,\dots,C_N\}D={(x(1),y(1)),(x(2),y(2)),…,(x(m),y(m))},y(j)∈{C1​,C2​,…,CN​}.OvO将这N个类别两两配对,从而产生N(N−1)/2N(N-1)/2N(N−1)/2个二分类任务.训练完成后,在测试阶段将获得N(N−1)/2N(N-1)/2N(N−1)/2个分类结果,最终结果按照被预测得最多的类别作为最终分类结果.

OvR

OvR每次将一个类的样例作为正例,所有其他类的样例作为反例来训练N个分类器,在测试时如果仅有一个分类器预测为正类,则对应的类别标记作为最终的分类结果.若有多个分类器预测为正类,则通常考虑各个分类器的预测置信度,选择置信度最大的类别标记作为分类结果.

MvM

MvM是每次将若干个类作为正类,若干个其他类作为反类,OvO和OvR都是MvM的特殊情形. MvM正反类构造最常见的技术是"纠错输出码"(Error Correcting Output Codes, ECOC).
"纠错输出码"主要将N个类进行M次划分,每次划分都有正类和反类,从而形成了二分类训练集,这样会生成M个分类器.使用这M个分类器进行预测,得到一个预测组成的编码,将这个编码与每个类别的编码进行比较,返回其中距离最小的类别作为最终预测结果.

以上分类方法都是基于二分类,有没有一种可以直接分类的方法呢?
